**Collocated with the [9th Colombian HPC Summer School](/summer_school_9/)** 

## Objective
<p style="text-align: justify; text-justify: inter-word;">
Quantum Day 2026 brings together Team Xanadu and Amazon Braket for an introductory, real-platform training on quantum computing. Participants will explore the visions of leading quantum hardware providers, learn how to access live cloud devices, and build practical workflows using modern quantum software.
</p>

## Date/Time
<p style="text-align: justify; text-justify: inter-word;">
June 19, 2025. <br/>
8:30 AM (COT) - 5:00 PM (COT)
</p>

## Program Structure
<p style="text-align: justify; text-justify: inter-word;">
The program consists of two technical sessions that blend platform vision, hardware insight, and hands-on experimentation:
</p>

### Session 1: Quantum Hardware and Cloud Platforms
<p style="text-align: justify; text-justify: inter-word;">
An overview of the current quantum ecosystem with a focus on Amazon Braket and the provider landscape. Participants will learn how leading hardware suppliers deliver superconducting, trapped-ion, and photonic access through cloud platforms, and how real quantum experiments are executed today.
</p>

**Topics Covered**

* Quantum computing vision from hardware and platform providers
* Amazon Braket architecture, service model, and supported backends
* Device characteristics: noise, connectivity, and execution flow
* Real-device access patterns and hybrid experiment design
* Practical demo: build and run a simple circuit on a cloud backend

| Time | Topic |
| :---: | :---: |
| 8:30 | Welcome and quantum computing overview |
| 8:45 | The role of cloud platforms in quantum research |
| 9:00 | Amazon Braket and provider ecosystem |
| 9:20 | Hardware case studies: superconducting, trapped-ion, photonic |
| 9:40 | Hands-on demo: submitting a circuit to Braket |
| 10:10 | Understanding noise, fidelity, and results |
| 10:30 | Coffee Break |
| 11:00 | Guided lab: simulator vs. real backend comparison |
| 11:45 | Review and open discussion |
| 12:00 | Session 1 wrap-up |

### Session 2: Quantum Software with Xanadu and PennyLane
<p style="text-align: justify; text-justify: inter-word;">
A practical introduction to quantum programming using Team Xanadu’s PennyLane library and hybrid quantum-classical workflows. Attendees will see how to write, optimize, and execute quantum circuits with real platform integration.
</p>

**Topics Covered**

* PennyLane fundamentals and hardware-agnostic programming
* Hybrid quantum-classical algorithms and variational circuits
* Connecting software to quantum backends through Braket and Xanadu tools
* Best practices for experiment design and platform selection

| Time | Topic |  |
